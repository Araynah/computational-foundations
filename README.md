# Computational & Mathematical Foundations

Coursework from the mathematical and algorithmic core of my Data Science degree at **Michigan State University**. These courses form the theoretical backbone behind modern machine learning and data science — covering linear algebra, optimization, statistical methods, and algorithm design.

---

## Courses

### CMSE 381 — Matrix Algebra with Computational Applications
**Semester:** Fall 2025

Applied linear algebra implemented computationally in Python. Focused on building intuition for the matrix operations that underlie machine learning methods like PCA, linear regression, and neural networks.

**Topics covered:**
- Matrix operations, decompositions (LU, QR, SVD)
- Eigenvalues and eigenvectors
- Positive semi-definite matrices
- Numerical methods and computational stability
- Applications to data science (dimensionality reduction, least squares)

**Tools:** Python, NumPy, Jupyter Notebook

---

### CMSE 382 — Fundamentals of Data Science Methods
**Semester:** Fall 2025

Mathematical foundations of core data science methods. Heavy emphasis on deriving and understanding the theory behind algorithms rather than just applying them.

**Topics covered:**
- Gradient descent and optimization
- Positive semi-definite matrices and convexity
- Probability distributions and statistical inference
- Regularization (L1/L2)
- Mathematical foundations of ML models

**Tools:** Python, NumPy, mathematical derivations (by hand and computational)

---

### CSE 331 — Algorithm Engineering & Design
**Semester:** Fall 2025

Design, analysis, and implementation of fundamental algorithms and data structures. Focused on correctness, efficiency, and complexity analysis.

**Topics covered:**
- Time and space complexity (Big-O analysis)
- Sorting and searching algorithms
- Dynamic programming
- Graph algorithms (BFS, DFS, shortest paths)
- Greedy algorithms and divide-and-conquer

**Tools:** Python, C++

---

## Why This Matters

Understanding the math behind the tools separates engineers who can apply models from those who can debug, adapt, and improve them. These courses build intuition for:

- **Why** gradient descent converges (or doesn't)
- **How** matrix decompositions power PCA and SVD-based recommenders
- **When** an algorithm is efficient enough to scale
